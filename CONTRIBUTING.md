# Contributing to LuECE Advisor 🤝

We love contributions! Whether you're an ECE student or from another department at Lucknow University, your help makes this tool better for everyone.

## 🚀 How to Add Your Branch

If you want to add data for a different branch (e.g., CSE, ME, Civil), follow these steps:

### 1. Scrape Department Data
We use an AI-powered scraper to extract information from the University's official portal. Run the following command with the link to your department's detail page:

```bash
# Example for a specific department page
npm run scrape "https://udrc.lkouniv.ac.in/Department/DepartmentDetail/History?dept=XX" --structure
```

> [!TIP]
> Replace `XX` with your department's ID. The `--structure` flag will automatically use AI to format the raw text into the required `ECE_DATA` structure.

### 2. Update the Knowledge Base
1.  Copy the output generated by the scraper.
2.  Open `src/lib/data.ts`.
3.  Add your branch data to the file or update existing fields.

### 3. Add Cutoff Data
If you have a PDF link for your branch's cutoffs, use the PDF extractor:

```bash
node scripts/extract-pdf.mjs "https://link-to-counselling-pdf"
```
Copy the structured output into `src/lib/cutoff.ts`.

## 🛠️ Contribution Workflow

1.  **Fork** the repository.
2.  **Create a branch** for your feature: `git checkout -b feature/add-cse-data`.
3.  **Commit** your changes: `git commit -m "feat: add CSE department data"`.
4.  **Push** to your fork: `git push origin feature/add-cse-data`.
5.  **Open a Pull Request** describing what you added!

## 📜 Guidelines
-   **No Hallucinations**: Ensure all data added is sourced from official university links.
-   **Keep it Premium**: Maintain the warm-toned, minimalist design style described in `lovable/DESIGN.md`.
-   **Test the API**: If you change the API routes, run `npm run dev` and test the chat to ensure streaming and RAG are still working.

Thank you for helping the LU community! 🌟
